# ATOTP-autenticator

In this repository You can see application that generate OTP (One-Time Password) codes.
This application is one of possible realisations of ATOTP (Address-based Time-based One-Time Password) algorithm.
ATOTP differs TOTP (Time-based OTP) in that it generates an OTP based not only on the time, but also on the destination address.

## System requirements

OS must be:

* Android
* IOS
* Linux
